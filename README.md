# api-project

# Description

Json Api utilizing mongoose and express.

Beautiful data extracted from https://hassavocadoboard.com/


**Endpoints**
POST	/avocados
GET	/avocados/:id
GET	/avocados/:id
PUT	/avocados/:id
DELETE	/avocados/:id


**JSON Schema**
-  Date: { type: String },
-  AveragePrice: { type: Number },
-  type: { type: String },
-  year: { type: Number },
-  region: { type: String },
